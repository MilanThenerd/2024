\doxysubsection{Detailed Description}
The \doxylink{classDeviceState}{Device\+State} class holds information about the device state. 

This class serves as the base for all states used by a \doxylink{classDevice}{Device}.

The class contains methods to retrieve the type and description of the state.

This abstract class holds the type and description of a device state. Derived classes will define specific behaviors for different states.
