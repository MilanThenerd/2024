\doxysubsection{Detailed Description}
The \doxylink{classSensor}{Sensor} class serves as a base for all sensor types in the system. 

This class represents a generic sensor in a smart home system.

This class allows for the addition and removal of notification commands and provides pure virtual methods for performing actions and triggering events specific to the type of sensor.

The \doxylink{classSensor}{Sensor} class inherits from \doxylink{classDevice}{Device} and provides an interface for sensors that can trigger notifications and perform actions. It manages a collection of \doxylink{classNotifyCommand}{Notify\+Command} objects associated with the sensor.
