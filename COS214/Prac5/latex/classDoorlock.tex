\doxysubsection{Detailed Description}
The \doxylink{classDoorlock}{Doorlock} class represents a simulated door lock device. 

This class simulates a door lock and inherits from the \doxylink{classDevice}{Device} class.

This class provides concrete implementations for the perform\+Action and print\+State methods from the base \doxylink{classDevice}{Device} class, enabling the simulation of a door lock\textquotesingle{}s behavior.

The \doxylink{classDoorlock}{Doorlock} class provides specific functionality for a door lock device, including actions to manipulate its state (lock, unlock, etc.), and printing the current state.
