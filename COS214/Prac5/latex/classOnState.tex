\doxysubsection{Detailed Description}
The \doxylink{classOnState}{On\+State} class simulates the state of a device being turned on. 

This class represents the on state of a device.

This class is part of the state pattern and provides the specific state of a device when it is active. The type is set to "{}\+Power"{} and the description is set to "{}\+On."{}

The \doxylink{classOnState}{On\+State} class inherits from \doxylink{classDeviceState}{Device\+State} and is used to indicate that a device is currently powered on. It defines the type and description associated with this state.
