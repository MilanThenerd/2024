\doxysubsection{Detailed Description}
The \doxylink{classLegacyThermostat}{Legacy\+Thermostat} class simulates a simple thermostat device. 

This class represents a legacy or "{}dumb"{} thermostat device.

This class allows you to manually control the temperature by increasing or decreasing it. It holds a temperature value that can be adjusted through its methods.

The \doxylink{classLegacyThermostat}{Legacy\+Thermostat} class simulates the behavior of an old-\/fashioned thermostat that manually adjusts the temperature without any smart capabilities.
