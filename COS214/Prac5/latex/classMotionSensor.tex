\doxysubsection{Detailed Description}
The \doxylink{classMotionSensor}{Motion\+Sensor} class simulates a motion sensor in a smart home system. 

This class represents a motion sensor that can trigger other devices.

This class allows for the detection of motion and can trigger specific actions in other devices when motion is detected. It provides methods to perform actions and print its state.

The \doxylink{classMotionSensor}{Motion\+Sensor} class inherits from the \doxylink{classSensor}{Sensor} class and simulates a motion sensor that detects motion and can trigger associated devices to perform actions.
