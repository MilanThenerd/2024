\doxysubsection{Detailed Description}
The \doxylink{classRoom}{Room} class manages a collection of devices in a smart home environment. 

This class represents a room in a smart home system.

This class allows users to interact with multiple devices in a specific room, facilitating actions such as adding or removing devices and controlling their states.

The \doxylink{classRoom}{Room} class inherits from \doxylink{classDevice}{Device} and is used to manage multiple devices within a specific room. It provides functionality to add, remove, and perform actions on the devices contained within the room.
