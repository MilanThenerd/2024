\doxysubsection{Detailed Description}
The \doxylink{classSmartHomeSystem}{Smart\+Home\+System} class manages the overall functionality of a smart home. 

This class represents the smart home system, managing devices, rooms, and sections.

This class holds vectors and maps of devices, rooms, sections, and routines, enabling the organization and control of a smart home environment.

The \doxylink{classSmartHomeSystem}{Smart\+Home\+System} class serves as the main controller for a smart home, allowing users to manage devices, rooms, sections, and routines. It provides functionality to add and remove these entities and to run the system.
