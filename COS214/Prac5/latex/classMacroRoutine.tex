\doxysubsection{Detailed Description}
The \doxylink{classMacroRoutine}{Macro\+Routine} class represents a macro that can execute a sequence of commands. 

This class automates the execution of multiple commands as part of a macro routine.

This class allows users to group multiple \doxylink{classHomeCommand}{Home\+Command} objects and execute them in order, automating a series of tasks. It provides functionality for adding and removing commands from the macro.

The \doxylink{classMacroRoutine}{Macro\+Routine} class inherits from \doxylink{classHomeCommand}{Home\+Command} and allows multiple commands to be grouped together and executed in sequence. This is useful for automating a series of actions on devices.
