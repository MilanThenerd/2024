\doxysubsection{Detailed Description}
The \doxylink{classDevice}{Device} class manages the state and identification of a sensor. 

The \doxylink{classDevice}{Device} class serves as the base class for all sensors.

The class includes methods for performing actions, retrieving the device\textquotesingle{}s status, and handling state changes. It also works with different concrete states like \doxylink{classOnState}{On\+State}, \doxylink{classOffState}{Off\+State}, \doxylink{classLockedState}{Locked\+State}, and \doxylink{classOpenState}{Open\+State}.

This abstract class provides the common functionality for all devices, including state management, identification, and action execution. It works with various device states (\doxylink{classOnState}{On\+State}, \doxylink{classOffState}{Off\+State}, \doxylink{classLockedState}{Locked\+State}, \doxylink{classOpenState}{Open\+State}).
