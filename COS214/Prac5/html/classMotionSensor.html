Detailed Description

The MotionSensor class simulates a motion sensor in a smart home system.

This class represents a motion sensor that can trigger other devices.

This class allows for the detection of motion and can trigger specific actions in other devices when motion is detected. It provides methods to perform actions and print its state.

The MotionSensor class inherits from the Sensor class and simulates a motion sensor that detects motion and can trigger associated devices to perform actions.
