Detailed Description

The Doorlock class represents a simulated door lock device.

This class simulates a door lock and inherits from the Device class.

This class provides concrete implementations for the performAction and printState methods from the base Device class, enabling the simulation of a door lock's behavior.

The Doorlock class provides specific functionality for a door lock device, including actions to manipulate its state (lock, unlock, etc.), and printing the current state.
