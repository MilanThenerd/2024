Detailed Description

The LockedState class simulates the state of a device being locked.

This class represents the locked state of a device.

This class is part of the state pattern and provides the specific state of a device when it is locked. It sets the type and description to represent the locked condition of the device.

The LockedState class inherits from DeviceState and is used to simulate a device being in the locked state. When a device is in this state, its type is set to "Lock" and its description is "Locked."
