Detailed Description

The LockCommand class implements the command to lock a device.

This class represents the LockCommand, used to simulate locking devices such as the Doorlock.

This class is part of the command pattern implementation, sending a locking command to the device (receiver) to change its state to locked.

The LockCommand class inherits from the HomeCommand class and provides the functionality to lock a device by sending the appropriate command to the receiver (e.g., a Doorlock).
